# Product Images

Please add the following product images to the `public/products/` directory:

1. **plastic-crate.jpg** - Plastik kasa/kasa görseli
2. **waste-container.jpg** - Çöp konteyneri görseli
3. **plastic-pallet.jpg** - Plastik palet görseli
4. **industrial-basket.jpg** - Endüstriyel sepet görseli

These images will be displayed in the Products dropdown menu in the navigation bar.

## Recommended Image Specifications

- **Format**: JPG or PNG
- **Dimensions**: 400x300px (or similar 4:3 aspect ratio)
- **File size**: Under 200KB for optimal loading
- **Background**: Clean, preferably white or light background
- **Quality**: Professional product photography style
