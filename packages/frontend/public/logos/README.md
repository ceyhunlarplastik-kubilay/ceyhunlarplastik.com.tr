# Logo Klasörü

Bu klasöre şirket logosunu ekleyin:

## Gerekli Dosya

- **ceyhunlar.png** - Ceyhunlar Plastik şirket logosu

## Önerilen Özellikler

- **Format**: PNG (şeffaf arka plan için)
- **Boyutlar**: 400x160px veya benzer oran (genişlik:yükseklik = 5:2)
- **Dosya boyutu**: 50KB altında
- **Arka plan**: Şeffaf (transparent)
- **Kalite**: Yüksek çözünürlük, net kenarlar

Logo, navigasyon menüsünün "Kurumsal" dropdown'ında öne çıkan bölümde gösterilecektir.
